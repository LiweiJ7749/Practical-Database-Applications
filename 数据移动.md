# 练习一：使用sqlldr导入数据

## 1. 创建表

```sql
SYS@sales>
CREATE TABLE scott.animal_feeding (
animal_id NUMBER,
feeding_date DATE,
pounds_eaten NUMBER (5,2),
note VARCHAR2(80) );

SYS@sales>select * from scott.animal_feeding;
no rows selected

```

## 2. 书写数据文件

```sh
[oracle@ocpsales dbhome_1]$ cd /u01/demo
[oracle@ocpsales demo]$ vim animal_feeding.dat
# 粘贴数据文件
```

## 3.书写控制文件（重要）

```sh
[oracle@ocpsales demo]$ vim animal_feeding.ctl

## 以下是控制文件
# 0. LOAD 加载
# 1. INFILE 数据源
# 2. INTO TABLE 目标表，只能插入空表
# 3. () 指定数据格式
# 4. TRAILING NULLCOLS 允许为空值
# 5. OPTIONALLY ENCLOSED BY '"' 双引号为可选定界符

LOAD
  INFILE '/u01/demo/animal_feeding.dat'
  insert
  INTO TABLE scott.animal_feeding
    TRAILING NULLCOLS
    (
      animal_id       INTEGER EXTERNAL TERMINATED BY ',',
      feeding_date    DATE "dd-mon-yyyy" TERMINATED BY ',',
      pounds_eaten    DECIMAL EXTERNAL TERMINATED BY ',',
      note            CHAR TERMINATED BY ','
                         OPTIONALLY ENCLOSED BY '"'
    )


```

## 4. 插入到`sales`数据库中

### 1）改变环境变量

```sh
[oracle@ocpsales demo]$ echo $ORACLE_SID
db19c
[oracle@ocpsales demo]$ export ORACLE_SID=sales
[oracle@ocpsales demo]$ echo $ORACLE_SID
sales
```

### 2）使用`sqldlr`导入数据

```sh
[oracle@ocpsales demo]$ sqlldr scott/tiger control=animal_feeding.ctl

  28 Rows successfully loaded.
```
